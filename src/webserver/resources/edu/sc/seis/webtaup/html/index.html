TauP Web interface

TauP Online:

Calculate times, paths and more interactively.

Examples:

  • describe phase: /phase?distdeg=35&evdepth=100&phases=P,S
  • time: /time?distdeg=35&evdepth=100&phases=P,S
  • pierce: /path?distdeg=35&evdepth=100&phases=P,S&format=text
  • path: /path?distdeg=35&evdepth=100&phases=P,S&format=svg
  • wavefront: /wavefront?distdeg=35&evdepth=100&phases=P&format=svg
  • curve: /curve?evdepth=100&phases=P&format=svg
  • scatter: /time?scatter=200,20&distdeg=35&evdepth=100&phases=P,S,PoP,SOp&format=svg
